MNEMO - App Review Notes

Mnemo is a private memory capture and recall app for iPhone.

CURRENT BUILD SUMMARY
This build captures user-entered text, voice transcripts, and image OCR results, then stores the resulting memories locally on the device using SwiftData. Chat recall searches saved local memories and returns matching memories from the on-device store.

AI AND EXTRACTION STATUS
- Text capture: processed locally into a structured memory record.
- Voice capture: uses SFSpeechRecognizer and local audio capture. Simulator speech recognition may be unavailable; physical-device testing is recommended for voice.
- Image capture: uses Apple Vision VNRecognizeTextRequest for OCR.
- Recall: searches local saved memories and returns matching stored memories.
- Vector search: uses local SQLite plus deterministic placeholder embeddings.

The codebase contains routing hooks for Apple Foundation Models and a future MLX model, but this build does not bundle a production Foundation Models or MLX embedding model. No cloud LLM provider is configured in this build, and no external LLM requests are made in this build.

BACKUP
CloudKit backup uses the user's own iCloud account. Mnemo does not operate a server for backup storage. Current restore requires the local Keychain backup key on this device, so this build should not be described as lost-device recovery until that path is validated.

APP LOCK / BIOMETRICS
Optional App Lock uses iOS LocalAuthentication only, using Face ID, Touch ID, or the device passcode where available. Mnemo does not receive, store, or upload biometric data, biometric templates, passcodes, account credentials, or backend identity data.

PERMISSIONS
The app does not require microphone, camera, or photo library permissions to function.
These are requested contextually only when the user first uses each feature:
- Microphone: when user taps the voice capture button
- Camera/Photos: when user taps the image capture button
- Notifications: not requested during onboarding in this build. Future Memory Moments reminders are shown as not active.

TESTING THE APP
1. Complete onboarding.
2. In the main app, tap the compose button to capture a memory via text.
3. Type a memory and tap Review Memory, then Save Memory.
4. In Chat, ask about what was saved to test local recall.
5. Browse shows captured memories.
6. Settings shows privacy controls, backup/restore, inactive Sense feature labels, and delete-all-data controls.

SANDBOX / TEST ACCOUNT
No Mnemo account, sign-up, login, or test credentials are required. The app does not create a backend identity.
The app is usable without iCloud. Optional iCloud backup uses the reviewer's signed-in Apple ID/iCloud account only; Mnemo does not operate an account server.
